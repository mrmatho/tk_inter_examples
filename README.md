# TkInter and CustomTkinter examples

This repository exists to provide code examples of TkInter and CustomTkinter for use by Heathmont College VCE Software Development students. These are being used in conjunction with class instruction to support students wishing to use one or both of these libraries for their SAT interfaces.

 It is not meant to be a comprehensive guide, but rather a collection of examples that can be used as a reference as you build applications. Each example has both Tkinter and CustomTkinter versions, so you can see how the same interface can be built using both libraries.
